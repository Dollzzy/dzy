Html syntaxa
Syntaxa Html este destul de simpla 
< !DOCTYPE html>

< html lang="en">

    < head>

        < meta charset="UTF-8">
        < meta name="viewport" content="width=device-width, initial-scale=1.0">
        < title> aici adaugi titlul site-ului / pagini< /title >

    < /head >

< body >
                    
< /body >

< /html >
Cum codam?
In body se vor crea elementele adica acolo vei crea elementele de care ai nevoie si in hader se vor pune linkurile catre alte fisiere utile cum ar fi Css sau Javascript bine fisierul javascript poate fi pus si sup body intre < / body > si < / html >
Linkul css in html
Pentru a adauga un fisier Css in html vom scrie urmatorul lucru
< link rel="stylesheet" href="nume.css">
Unde scrie nume.css vom scrie numele fisierului css
Truc vs code
Daca scri link:css si apasati enter automat ti se va crea un link catre pagina css
De retinut
In aceasta syntaxa putem vedea ca avem cam multe elemente html dar pentru a nu scrie toate acestea si a pierde timp putem face urmatorul lucru
Trucuri vs code
Daca tastati ! intrun fisier html si apasati enter automat va aparea aceasta Syntaxa
