Clase Elemente
Elementelor li se pot atribui clase pentru a fi recunoscute de catre Css si Javascript mai usor exemplu : 
    < element    class=" aici scri un nume " >

    < / element >
Dupa ce scri un nume dupa care va fi recunoscut elementul il poti folosi in Css pentru al stiliza exemplu:
Html
< element    class="test" >
                buna
 < / element >
Css
    test {
        color: red;
    }
Rezultat
Buna
De ce punem . [punct] in fata unui element in Css? ei bine punem punct in fata elementului pentru ca Css sa stie ca elementul are o clasa si nu un ID.
Ce este un ID
Va recomand sa vedeti: ⚬ Id Element pentru mai multe informati si exemple
De ce nu merge?
De retinut daca scrieti clasa elementulului cu litere mici ca exemplu: 
< element    class="test" >
    buna
 < / element >
Atunci cand veti scrie numele dupa . [punct] in Css numele il veti scrie tot cu litere mici sau cu literere mari daca asa ati scris numele exemplu:
Css
.test {
      color: red;
}
            
.Test {
      color: red;
}
De aceasta este importatnt sa fi atent orice litera pe care o scri finca se poate sa nu fie recunoscut si codul dumneavoastra sa nu mearga
