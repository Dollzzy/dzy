Syntaxa
< object > < / object >
Prin acest exemplu putem vedea cum este syntaxa Html-ului cu < incepem sa creem un element html si cu > incheiem elementul ca exemplu < element > apoi vom rescrie acelasi element dar vom pune dupa < un slash / exemplu < / element > Dar care dintre ele este metoda corecta de scriere a unui element? nici una finca ambele se folosesc pentru a creea un singur element cu < element > < / element > creeem un singur element in pagina noastra html cu < element > noi ii zicem codului de unde sa inceapa elementul si cu < / element > ii zicem codului unde se termina elementul, tu in acest element poti scrie sau crea alt eleemnt si uite de aceasta ii bine ca avem nevoie de o incepere si de o incheiere ca in cazul in care vei crea mai multe elemente acesta sa stie unde se termina primul si unde se termina al doilea
