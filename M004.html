Darkweb
Dark web (în română, "webul întunecat") reprezintă o parte a internetului care nu este indexată și accesibilă prin motoarele de căutare obișnuite, cum ar fi Google, Bing sau Yahoo. Este o parte ascunsă a internetului care necesită software și configurări speciale pentru a fi accesată. Dark web-ul include site-uri web, forumuri, piețe online și alte resurse care nu sunt ușor de găsit sau accesat pentru utilizatorii obișnuiți.
Trafic darkweb
Deoarece traficul și conținutul de pe dark web sunt adesea criptate și ascunse, această parte a internetului a câștigat o reputație de a găzdui activități ilegale sau suspecte, cum ar fi vânzarea de droguri, arme, date personale furate, servicii de hacking, identități false și multe altele. Totuși, este important de menționat că nu întreaga activitate de pe dark web este ilegală sau neetică. Există și utilizatori care îl folosesc pentru a se proteja de supraveghere, pentru a accesa conținut cenzurat sau pentru a comunica în mod privat.
Darkweb deepweb
Dark web-ul este diferit de "deep web", care reprezintă conținutul care nu este indexat de motoarele de căutare, dar nu este neapărat ilegal sau ascuns. O mare parte a internetului este de fapt deep web, cum ar fi baze de date private, conturi bancare online, e-mailuri personale etc.
De retinut
Este important să fii conștient de potențialele riscuri și implicații legate de accesarea dark web-ului, iar dacă ești curios sau interesat să explorezi, trebuie să fii extrem de atent și să iei măsuri adecvate de securitate pentru a-ți proteja identitatea și dispozitivele.

Tor
TOR (The Onion Router) este un software de rețea care permite utilizatorilor să navigheze pe internet în mod anonim. Acesta a fost inițial dezvoltat de către Marinade Zalewski și apoi preluat de către Electronic Frontier Foundation (EFF) și continuat sub diverse proiecte. Scopul principal al rețelei TOR este să ofere confidențialitate și anonimat utilizatorilor prin redirecționarea traficului lor pe mai multe noduri sau servere, ascunzând astfel identitatea și locația reală a utilizatorului.

Reguli tor
daca accesati tor de pe windows este important sa nul deschideti in fullscreen finca unele site-uri vor rula pe fundal diferiti virusi sau vor incerca sa iti acceseze conturile
Onion
Linkurile Onion sunt adrese web specifice care se termină cu ".onion" și sunt utilizate pentru a accesa site-uri și servicii care sunt găzduite pe rețeaua Tor. Aceste site-uri nu pot fi accesate cu un browser web normal, cum ar fi Chrome sau Firefox, și necesită utilizarea unui browser specializat Tor, cum ar fi Tor Browser.
